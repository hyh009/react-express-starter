# I18n Development

Use this when setting up Google Sheet sync for frontend i18n.

## Local Config

Create this untracked file:

```txt
apps/web/scripts/i18n.config.json
```

Copy from:

```txt
apps/web/scripts/i18n.config.example.json
```

Shape:

```json
{
  "spreadsheetId": "your-google-spreadsheet-id",
  "sheetTitle": "Translations",
  "languages": ["en", "zh-TW"],
  "credentialsPath": "secrets/google-service-account.json"
}
```

- `spreadsheetId` is the Google Spreadsheet id from the URL.
- `sheetTitle` is the tab name. If omitted, the first tab is used.
- `languages` must match locale columns in the Sheet.
- `credentialsPath` points to the service account JSON.

## Secrets

Put the service account JSON under:

```txt
secrets/google-service-account.json
```

`secrets/` is ignored by git.

## Commands

```bash
pnpm --filter web run i18n:scan
pnpm --filter web run i18n:sync
pnpm --filter web run i18n:generate
```

- `i18n:scan` scans `tDefault()` calls.
- `i18n:sync` adds missing Sheet rows and updates the `default` column.
- `i18n:generate` reads the Sheet and writes `resources.generated.ts`.

Normal order:

```bash
pnpm --filter web run i18n:scan
pnpm --filter web run i18n:sync
pnpm --filter web run i18n:generate
```

Env fallback is also supported when `i18n.config.json` is not present:

```txt
I18N_SPREADSHEET_ID=...
GOOGLE_APPLICATION_CREDENTIALS=secrets/google-service-account.json
I18N_SHEET_TITLE=Translations
I18N_LANGUAGES=en,zh-TW
```
